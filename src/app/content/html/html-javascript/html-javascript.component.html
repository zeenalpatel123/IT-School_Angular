The HTML <script> Tag

The HTML <script> tag is used to embed or reference client-side JavaScript code within a webpage.

The <script> tag can include JavaScript code directly or link to an external file using the src attribute.

JavaScript is commonly used for tasks like image handling, validating form inputs, and updating content dynamically on a webpage.

JavaScript commonly selects HTML elements using the document.getElementById() method.

This JavaScript example inserts the text "Hello JavaScript!" into the HTML element that has the id set to "demo".

Example

<script >
document. getElementById( "demo"). innerHTML = "Hello JavaScript!";
</script >
