The <!DOCTYPE> Declaration

The  <!DOCTYPE>   declaration defines the document type and ensures that browsers render web pages correctly.

  • It must appear once at the top of the page, before any HTML tags.
  • It is not case-sensitive.
  • The  <!DOCTYPE>   declaration for HTML5 is:
<!DOCTYPE html>

HTML Headings

HTML headings are created using the   <h1>   To   </h6>   tags.

  • The  <h1>   represents the Most important heading
  • The  <h6>   represents the Least important heading
<h1> My First Heading </h1>

<h2> My First Heading </h2>

<h3> My First Heading </h3>
