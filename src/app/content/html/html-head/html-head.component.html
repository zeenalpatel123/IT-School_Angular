HTML - The Head Element

The HTML <head> element is a container for the following elements: <title>, <style>, <meat>, <link>, <scrpit>, and <base>.

The HTML <head> Element

The <head> element holds metadata about the document and is positioned between the <html> tag and the <body> tag.

HTML metadata provides information about the web page itself. This data isn't shown directly on the page.

Metadata usually includes details like the document's title, character encoding, linked stylesheets, scripts, and other descriptive information.

The HTML <title> Element

The <title> tag sets the title of the webpage. It should contain only text and appears in the browser tab or window title bar.

The <title> element is a mandatory part of every HTML document.

A page's title plays a key role in search engine optimization (SEO). Search engines use the title to help determine how pages are ranked in search results.

The <title> element:

  • defines a title in the browser toolbar
  • provides a title for the page when it is added to favorites
  • displays a title for the page in search engine-results

So, try to make the title as accurate and meaningful as possible!

A simple HTML document:

Example

<!DOCTYPE html>
<html >
<head >
  <title > A Meaningful Page Title </title >
</head >
<body >

The content of the document......

</body >
</html >

The HTML <style> Element

The <style> element is used to define style information for a single HTML page:

Example

<style >
  body { background-color : powderblue ; }
  h1 { color : red ; }
  p { color : blue ; }
</style >

The HTML <link> Element

The <link> tag specifies the connection between the current HTML document and an external resource.

The <link> element is commonly used to connect an external CSS stylesheet to the HTML document.

Example

<link rel ="stylesheet" href ="mystyle.css" >

The HTML <meta> Element

The <meta> tag is often used to define the character encoding, page description, keywords, author information, and viewport configuration for the document.

Metadata doesn’t appear on the webpage itself, but it’s used by browsers for handling content and refresh settings, by search engines for indexing, and by various web services for processing page information.

Examples

Define the character set used:

<meta charset ="UTF-8" >

Define keywords for search engines:

< meta name="keywords" content="HTML, CSS, JavaScript">

Define a description of your web page:

< meta name="description" content="Free Web tutorials">

Define the author of a page:

< meta name="author" content="mini">

Refresh document every 60 seconds:

< meta http-equiv="refresh" content="60">

Setting the viewport to make your website look good on all devices:

< meta name="viewport" content="width=device-width, initial-scale=1.0">

Example of tags:

Example

<meta charset ="UTF-8" >
<meta name ="description" content ="Free Web tutorials" >
<meta name ="keywords" content ="HTML, CSS, JavaScript" >
<meta name ="author" content ="John Doe" >

Setting The Viewport

The viewport refers to the portion of a web page that is visible to the user. Its size depends on the device—smaller on smartphones and larger on desktop screens.

You should include the following <meta> element in all your web pages:

Setting the viewport to make your website look good on all devices:

< meta name="viewport" content="width=device-width, initial-scale=1.0">

This tells the browser how to manage the page’s size and scale on different devices.

The width=device-width setting makes the page's width match the screen width of the user's device, which can differ across phones, tablets, and desktops.

The initial-scale=1.0 setting defines the default zoom level of the page when it initially loads in the browser.

The HTML <script> Element

The <script> element is used to define client-side JavaScripts.

The following JavaScript writes "Hello JavaScript!" into an HTML element with id="demo":

Example

<script >
function myFunction() {
  document. getElementById( "demo"). innerHTML = "Hello JavaScript!";
}
</script >

The HTML <base> Element

The <base> element specifies the base URL and/or target for all relative URLs in a page.

The <base> tag must have either an href or a target attribute present, or both.

There can only be one single <base> element in a document!

Example

<head >
<base href ="https://www.w3schools.com/" target ="_blank" >
</head >

<body >
<img src ="images/stickman.gif" width ="24" height ="39" alt ="Stickman" >
<a href ="tags/tag_base.asp" > HTML base Tag </a >
</body >

Chapter Summary

  • The <head> tag acts as a container for metadata, which provides information about the HTML document.
  • The <head> element is located between the opening <html> tag and the <body> tag in an HTML document.
  • The <title> tag is mandatory and specifies the title of the HTML document.
  • The <style> tag is used to include CSS style rules directly within a single HTML document.
  • The <link> element is commonly used to connect an external CSS stylesheet to the HTML document.
  • The <meta> tag is generally used to define the character encoding, describe the page content, list keywords, identify the author, and set viewport settings for responsive design.
  • The <script> element is used to define client-side JavaScripts
  • The <base> tag sets the base URL and/or default target for all relative links on the page.
