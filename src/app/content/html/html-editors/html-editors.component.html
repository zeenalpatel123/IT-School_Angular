HTML Editors

A simple text editor is all you need to learn HTML.

Learn HTML with Notepad or TextEdit

  • You can create and edit web pages using professional HTML editors.
  • However, when learning HTML, we recommend using a simple text editor like Notepad (on Windows) or TextEdit (on Mac).
  • Using a basic text editor helps you understand HTML structure better and improves your coding skills.
  • Follow the steps below to create your first web page using Notepad or TextEdit.

Step 1: Launch Notepad on Windows

  • For Windows 10 or later:
    1. Click the Start Menu (Windows icon in the bottom left corner).
    2. Type Notepad in the search bar and press Enter.
  • For Windows 7 or earlier:
    1. Click Start > All Programs > Accessories.
    2. Select Notepad from the list.

Step 1: Open TextEdit (Mac)

  1. Open Finder > Applications > TextEdit.
  2. Adjust some settings to ensure your files are saved correctly:
    • Go to Preferences > Format and select "Plain Text".
    • Under the "Open and Save" tab, check the box that says "Display HTML files as HTML code instead of formatted text".
  3. Now, open a new document and start writing your HTML code.

Step 2: Write Some HTML

Write or copy the following HTML code into Notepad:

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>

<p>My first paragraph.</p>

</body>
</html>

Step 3: Save the HTML Page

Save the file to your computer by selecting File > Save As from the Notepad menu.

Give the file the name "index.htm" and choose UTF-8 as the encoding, as it is the recommended encoding for HTML files.

Tip: You can use either .htm or .html as file extension. There is no difference; it is up to you.

Step 4: Open the HTML File in Your Browser

Locate the saved HTML file on your computer and open it in your preferred web browser.

  • Double-clicking the file, or
  • Right-clicking the file and selecting "Open with", then choosing your browser.
