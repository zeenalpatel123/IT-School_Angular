HTML Attributes

HTML attributes provide additional information about HTML elements.

HTML Attributes

  • All HTML elements can have attributes.
  • Attributes provide extra information about an element.
  • They are always placed inside the start tag.
  • Attributes are typically written in name/value pairs, such as: name="value".

The href Attribute

The <a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to:

Example

<a href="https://www.google.com">Visit Google</a>

The src Attribute

The <a> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed:

Example

<img src="doll_img.jpg">
Try Now

There are two ways to specify the URL in the src attribute:

1. Absolute URL - Links to an external image that is hosted on another website. Example: src="https://www.w3schools.com/images/doll_img.jpg".

Notes:External images might be under copyright. If you do not get permission to use it, you may be in violation of copyright laws. In addition, you cannot control external images; it can suddenly be removed or changed.

2. Relative URL- Links to an image that is hosted within the website. Here, the URL does not include the domain name. If the URL begins without a slash, it will be relative to the current page. Example: src="doll_img.jpg". If the URL begins with a slash, it will be relative to the domain. Example: src="/images/doll_img.jpg".

Tip:It is almost always best to use relative URLs. They will not break if you change domain.

The width and height Attributes

The <img> tag should also contain the widht and height attributes, which specify the width and height of the image (in pixels):

Example

<img src="doll_img.jpg" widht= "500" height= "600">

The alt Attribute

The required alt attribute for the <img> tag specifies an alternate text for an image, if the image for some reason cannot be displayed. This can be due to a slow connection, or an error in the src attribute, or if the user uses a screen reader.

Example

<img src="doll_img.jpg" alt= "Beautiful Doll ">

The style Attribute

The style attribute is used to add styles to an element, such as color, font, size, and more.

Example

<p style="color:red;">This is a red paragraph.</p>

The lang Attribute

You should always include the lang attribute inside the <html> tag, to declare the language of the Web page. This is meant to assist search engines and browsers.

The following example specifies English as the language:

Example

<!DOCTYPE html>
<html lang="en">
<body>
...
</body>
</html>
Try Now

Country codes can also be added to the language code in the lang attribute. So, the first two characters define the language of the HTML page, and the last two characters define the country.

The following example specifies English as the language and United States as the country

Example

<!DOCTYPE html>
<html lang="en-US">
<body>
...
</body>
</html>

The title Attribute

The title attribute defines some extra information about an element.

The value of the title attribute will be displayed as a tooltip when you mouse over the element

Example

<p title="I'm a tooltip">This is a paragraph.</p>
Try Now

We Suggest: Always Use Lowercase Attributes

The HTML standard does not require lowercase attribute names.

The title attribute (and all other attributes) can be written with uppercase or lowercase like title or TITLE.

However, ITC recommends lowercase attributes in HTML, and demands lowercase attributes for stricter document types like XHTML.

Note: At W3Schools we always use lowercase attribute names.

We Suggest: Always Quote Attribute Values

The HTML standard does not require quotes around attribute values.

However, ITC recommends quotes in HTML, and demands quotes for stricter document types like XHTML.

Good:

<a href="https://www.google.com/html/">Visit our Google</a>
Try Now

Bad:

<a href=https://www.google.com/html/>Visit our Google</a>

Sometimes you have to use quotes. This example will not display the title attribute correctly, because it contains a space:

Example

Note: At IT-Schools we always use quotes around attribute values.

Single or Double Quotes?

Double quotes around attribute values are the most common in HTML, but single quotes can also be used.

In some situations, when the attribute value itself contains double quotes, it is necessary to use single quotes:

Example

<p title='Robart "ShotGun" Nelson'>

Or vice versa:

Example

<p title="Robart 'ShotGun' Nelson">

Chapter Summary

  • Every HTML element can include attributes to give extra information.
  • The href attribute in the <a> tag sets the web address the link should open.
  • The src attribute in the <img> tag shows the location of the image file.
  • The width and height attributes of <img> define the image size in pixels.
  • The alt attribute in <img> provides a backup text if the image can’t be displayed.
  • The style attribute is used to apply custom styles like text color, fonts, sizes, and more.
  • The lang attribute in the <html> tag declares the language used in the web page.
  • The title attribute adds a tooltip or hint that appears when hovering over an element.
