<div> as a container

The <div> element is often used to group sections of a web page together.

Example

<div >
  <h2 > United States </h2 >
  <p > United States is the capital city of England. </p >
  <p > United States has over 9 million inhabitants. </p >
</div >

Center align a <div> element

To center a <div> element that doesn't span the full width of its container, you can set its CSS margin property to auto.

Example

<style >
div {
  width :300px ;
  margin :auto ;
}
</style >

There are different methods for aligning elements side by side, all include some CSS styling. We will look at the most common methods:

Float

Although the CSS float property wasn't initially designed to position <div> elements next to each other, it has long been used for creating side-by-side layouts.

The CSS float property helps with layout and formatting by allowing elements to be aligned horizontally instead of stacking vertically.

Example

<style >
.mycontainer {
  width :100% ;
  overflow :auto ;
}
.mycontainer div {
  width :33% ;
  float :left ;
}
</style >

Inline-block

When you set a <div> element’s display property to inline-block instead of block, it removes the automatic line breaks, allowing multiple <div>s to appear side by side rather than stacking vertically.

Example

<style >
div {
  width : 30% ;
  display : inline-block ;
}
</style >

flex

The CSS Flexbox Layout Module was introduced to make it easier to design flexible responsive layout structure without using float or positioning.

To make the CSS flex method work, surround the <div> elements with another <div> element and give it the status as a flex container.

Example

<style >
.mycontainer {
  display : flex ;
}
.mycontainer > div {
  width :33% ;
}
</style >

Grid

The CSS Grid Layout Module provides a layout system based on rows and columns, simplifying web page design by eliminating the need for floats or manual positioning.

It’s similar to Flexbox, but Grid allows you to create multiple rows and control the placement of each row independently.

Using CSS Grid involves wrapping your <div> elements inside a parent <div> set as a grid container, and you need to define the width for each column in the layout.

Example

<style >
.grid-container {
  display : grid ;
  grid-template-columns : 33% 33% 33% ;
}
</style >
